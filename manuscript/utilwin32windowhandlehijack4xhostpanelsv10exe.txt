##  Util - Win32 Window Handle Hijack (4x host panels) v1.0.exe 

Using the exact same control used in [Util - Win32 Window Handle Hijack (simple) v1.0](http://diniscruz.blogspot.co.uk/2012/11/util-win32-window-handle-hijack-simple.html) here is a 4x Panel version of it (i.e. using 2 rows and 2 columns)

This allows for the _hijack _of a multiple Windows into a common (external) process/GUI

You can download this 1Mb tool from [Util - Win32 Window Handle Hijack (4x host panels) v1.0.exe](https://dl.dropbox.com/u/81532342/O2Platform%20Tools/Windows/Windows_Messages/Util%20-%20Win32%20Window%20Handle%20Hijack%20%284x%20host%20panels%29%20v1.0.exe)  


When you start the tool you will see 4 _Target _Icons, which represent the 4 _Hijack areas _available:
